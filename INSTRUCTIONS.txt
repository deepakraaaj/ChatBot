
# AI Backend - Running Instructions

## 1. Prerequisites
- Python 3.10+
- MySQL/MariaDB (Ensure it's running)
- Docker (Optional, for containerized run)

## 2. Setup (One-Time)
1.  **Create Virtual Environment:**
    ```bash
    python3 -m venv venv
    ```

2.  **Install Dependencies:**
    ```bash
    # Activate venv first (see below)
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3.  **Configure Environment:**
    - Copy `.env.example` to `.env` and fill in API keys/DB creds.
    - Run migrations/seed data if needed: `python init_db.py`

## 3. Daily Workflow

### ➤ Step 1: Switch to Virtual Environment (venv)
Always run this before running any other commands:
```bash
source venv/bin/activate  # Linux/Mac
# .\venv\Scripts\activate # Windows
```

### ➤ Step 2: Run the Backend Server
Starts the API at `http://localhost:8000`.
```bash
uvicorn app.api.main:app --reload
```

### ➤ Step 3: Run the Streamlit UI
Starts the frontend chat interface (usually at `http://localhost:8501`).
```bash
streamlit run streamlit_app.py
```

## 4. Verification
Run the verification script to check health and functionality:
```bash
python verify_backend.py
```
